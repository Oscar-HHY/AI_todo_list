# AI Todo List Assistant

An AI-powered task management assistant built with Python and Streamlit.

# Features

* Add and manage daily tasks
* AI assistant interaction using LLM
* Persistent local task storage
* Secure API key management

# Tech Stack

* Python
* Streamlit
* OpenAI-compatible API (DeepSeek)
* SQLite

# Project Structure

AI_todo_list
├── app.py
├── requirements.txt
├── .gitignore

# Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
